Message¶

class Message(**data)[source]¶

Bases: BaleObject

Represents a message in Bale messenger with all its metadata.

This class contains references to the chat the message belongs to, the sender, the message content, and related messages such as quoted replies or previous messages. Date fields are stored as Unix timestamps in milliseconds.

Parameters:

  • chat (Chat)

  • sender_id (int)

  • date (int)

  • message_id (int)

  • content (MessageContent)

  • quoted_replied_to (QuotedMessage | None)

  • previous_message (OtherMessage | None)

  • next_message (OtherMessage | None)

  • replied_to (Message | None)

  • extra_data (Any)

chat: Chat¶

The chat object that this message belongs to.

sender_id: int¶

The unique identifier of the sender (user ID).

date: int¶

Timestamp (in milliseconds) when the message was sent.

message_id: int¶

Unique identifier for this message within the chat.

content: MessageContent¶

The content of the message, which can be text, media, or other types.

quoted_replied_to: Optional[QuotedMessage]¶

If this message is a reply, the quoted message it replies to.

May not have the chat reference set initially, so it will be attached automatically.

previous_message: Optional[OtherMessage]¶

The message that was sent immediately before this one (if known).

next_message: Optional[OtherMessage]¶

The message that was sent immediately after this one (if known). Excluded from serialization because it’s used only internally.

replied_to: Optional[Message]¶

The full Message object that this message replies to (if available). Set automatically by attach_chat_to_reply validator.

property text: str | None¶

Returns the textual content of the message if available.

If the message content is not text-based, returns None.

attach_chat_to_reply()[source]¶

Ensures that if this message quotes another message without a chat assigned, the chat from this message is attached to it.

Also sets the replied_to field to the full replied message for easier access.
